# 🌍 115-earthscience-summer

> **北市大暑期地球科學課程計畫** | 2026/7/17 起，共 6 週

---

## 📌 專案簡介

本專案是 **台北市大學暑期地球科學課程**，專為 **中小學現職教師** 設計，結合 **地震學、地球物理** 與 **AI/Agent 工具**，目標是讓教師能將地震知識與數位工具應用在教學中，並產出可帶回學校的教學作品。

---

## 🎯 核心目標

1. **建立基礎認知**：讓教師理解地震學與地球物理的核心概念（如震源定位、地震波、震級等）。
2. **工具熟練應用**：學會使用 **Google GWS、GitHub、Hugging Face、Vercel、GAS、opencode、Hermes agent** 等工具。
3. **教學轉化**：將地震知識與工具轉化為 **中小學適合的教學活動**。
4. **產出教學作品**：每週完成一個可評核的作業，最終產出一份可帶回學校的教學作品。

---

## 📅 6 週課程概覽

| 週次 | 主題 | 核心工具 | 可評核作業 |
|------|------|----------|-------------|
| **Week 1** | 工具環境建置、地震學概論 | Google GWS、GitHub、Hugging Face、Vercel、GAS | ✅ 完成個人工具環境建置 |
| **Week 2** | 地震動、加速度、震級 | 手機感測器、Google Sheets、IRIS ground-shaking | ✅ 完成簡易震動觀測活動 |
| **Week 3** | P-S 到時、震源定位 | IRIS 震源定位、Python、GitHub | ✅ 完成震源定位教學範例 |
| **Week 4** | 地震波形分析 | jAmaSeis、Hugging Face Dataset | ✅ 完成波形分析小報告 |
| **Week 5** | 地震防災（規模、震級、建物效應） | IRIS Magnitude/Intensity、Vercel、GAS | ✅ 完成地震防災互動網頁 |
| **Week 6** | 野外實察、成果整合 | 地震儀/手機感測器、IRIS 即時資料 | ✅ 完成期末教學作品發表 |

## 🎓 如何開始？

1. **閱讀 [`course_plan.md`](./course_plan.md)**：了解完整的課程安排與目標。
2. **參考 [`學習指引.md`](./學習指引.md)**：取得學習建議、工具準備清單和資源連結。
3. **設定工具環境**：依照 Week 1 的內容，建立個人工具帳號（Google、GitHub、Hugging Face、Vercel）。
4. **跟著課程進度**：每週完成指定的作業，並思考如何應用在教學中。
