# 🌍 115-earthscience-summer 學習指引

> **課程名稱**：北市大暑期地球科學課程（地震學與地球物理 + AI/Agent 工具輔助教學實作）  
> **目標對象**：中小學現職教師  
> **課程期間**：2026/7/17 起，共 6 週（每週四次，每次 4 小時）  
> **核心目標**：學會使用地震資料、工具與 AI，設計可帶回學校的教學活動

---

## 📌 學習前的準備

### 1️⃣ **心態準備**
- **動手做**：這門課強調「實作」，每週都有具體的工具操作或教學設計任務。不要只聽講，要親自動手嘗試。
- **不怕錯**：工具（如 Git、Python、GAS）可能會有錯誤，遇到問題時先自己尋找解決方案（Google、官方文件、同學討論）。
- **教學導向**：時刻思考「如何把這個知識或工具帶回學校教學」。
- **持續學習**：6 週的課程只是起點，真正的學習在課後持續實踐。

### 2️⃣ **工具準備**
在 **Week 1** 之前，建議先完成以下工具的基礎設定：

| 工具 | 用途 | 準備方式 |
|------|------|----------|
| **Google 帳號** | GWS、GAS、Sheets、Forms、Drive | 確認有可用的 Google 帳號（建議使用學校或個人常用帳號） |
| **GitHub 帳號** | 程式碼版本控制、教材分享 | [註冊 GitHub](https://github.com/join) |
| **Hugging Face 帳號** | AI 模型、資料集 | [註冊 Hugging Face](https://huggingface.co/join) |
| **Vercel 帳號** | 部署網頁（如教學網站） | [註冊 Vercel](https://vercel.com/signup) |
| **VS Code** | 程式碼編輯器 | [下載安裝](https://code.visualstudio.com/) |
| **Git** | 版本控制工具 | [下載安裝](https://git-scm.com/downloads) |
| **Python** | 程式語言（Week 3 會用到） | [下載安裝](https://www.python.org/downloads/)（建議 Python 3.10 以上） |
| **Node.js** | JavaScript 運行環境（如需使用） | [下載安裝](https://nodejs.org/) |

### 3️⃣ **背景知識補充**
如果你對以下主題不熟悉，建議在課前先閱讀或觀看相關資源：
- **地震學基礎**：
  - 地震的成因（板塊運動、斷層）
  - 震級（Magnitude）與強度（Intensity）的差異
  - P 波、S 波、地震波的傳播
  - 震源（Hypocenter）與震央（Epicenter）
  - 旅行時間曲線（Travel-time curve）
- **地球物理基礎**：
  - 地球的內部結構（地殼、地函、地核）
  - 地震儀（Seismometer）的工作原理
- **工具基礎**：
  - Git 的基本概念（Repository、Commit、Push、Pull）
  - Google Apps Script（GAS）的基礎語法
  - Python 的基礎語法（如變數、迴圈、函式）

**推薦資源**：
- [IRIS 地震學教學資源](https://www.iris.edu/hq/inclass)（英文，但有豐富的互動工具）
- [台灣地震科學中心](https://www.earthquake.tw/)（中文，包含台灣地震資料）
- [GitHub 入門教學](https://guides.github.com/activities/hello-world/)（15 分鐘快速上手）
- [Python 入門教學](https://www.learnpython.org/)（互動式學習）

### 🔹 **Week 2：地震動與震級**
**主題**：地震動、加速度、地震動與震級

#### 🎯 學習目標
1. 了解地震動（Ground Motion）的基本概念（加速度、速度、位移）。
2. 學會使用手機感測器或專業地震儀記錄地震動。
3. 了解震級（Magnitude）的定義與計算方式。
4. 使用 IRIS 的 ground-shaking 活動進行教學設計。

#### 📚 學習重點
- **地震動**：
  - 加速度（Acceleration）、速度（Velocity）、位移（Displacement）
  - PGA（Peak Ground Acceleration）、PGV（Peak Ground Velocity）
  - 地震動對建築物的影響
- **震級**：
  - 里氏震級（Richter Scale）
  - 矩震級（Moment Magnitude Scale, Mw）
  - 震級與地震能量的關係
- **工具**：
  - 手機感測器（如 Android 的 Phyphox、iOS 的 SensorLog）
  - Google Sheets（資料分析）
  - IRIS ground-shaking 活動

#### 🛠️ 實作任務（作業2）
- [ ] 使用手機或專業地震儀記錄一次地震動（或模擬地震動）。
- [ ] 繪製一張地震動的圖表（例如：加速度 vs. 時間）。
- [ ] 完成一份簡單的教學設計（例如：如何用手機感測器教地震動）。
- [ ] 撰寫 200 字說明：這個活動如何幫助學生理解震級？

#### 💡 學習建議
- 如果沒有地震儀，可以使用手機的感測器 App（如 Phyphox）來模擬地震動。
- 多嘗試 IRIS 的 ground-shaking 活動，了解不同震級的地震動差異。
- 思考如何將地震動的概念簡化，讓中小學生能理解。

### 🔹 **Week 3：震源定位**
**主題**：P-S 到時、旅行時間曲線與震源定位

#### 🎯 學習目標
1. 了解 P 波（Primary Wave）和 S 波（Secondary Wave）的差異。
2. 學會使用 P-S 到時（S-P Time）來估算震源距離。
3. 了解旅行時間曲線（Travel-time Curve）的原理。
4. 使用 IRIS 的震源定位活動進行教學設計。

#### 📚 學習重點
- **地震波**：
  - P 波（縱波，壓縮波）
  - S 波（橫波，剪切波）
  - P 波和 S 波的速度差異
- **震源定位**：
  - P-S 到時（S-P Time）的計算
  - 旅行時間曲線的原理
  - 三角定位法（Triangulation）
- **工具**：
  - IRIS 震源定位活動
  - Python（資料分析）
  - GitHub（分享教材）

#### 🛠️ 實作任務（作業3）
- [ ] 使用三個測站的地震波資料，繪製 P 波和 S 波的到時表。
- [ ] 使用 P-S 到時估算震源距離。
- [ ] 繪製震源定位圖（例如：在地圖上標出震源位置）。
- [ ] 完成 GitHub 程式碼或 Google Sheets 連結。
- [ ] 提出一個教學提問（例如：如何用三個測站定位震源？）。

#### 💡 學習建議
- 如果對 Python 不熟悉，可以先學習基本的資料分析（如使用 `pandas` 讀取 CSV 文件）。
- 多嘗試 IRIS 的震源定位活動，了解不同測站的資料如何影響定位結果。
- 思考如何將震源定位的概念簡化，讓中小學生能理解。

### 🔹 **Week 5：地震防災**
**主題**：規模、震級、建物效應與土壤放大效應

#### 🎯 學習目標
1. 了解地震規模（Magnitude）和震級（Intensity）的差異。
2. 了解建物效應（Building Effects）和土壤放大效應（Site Amplification）。
3. 學會使用 IRIS 的 Magnitude/Intensity 資源。
4. 設計地震防災的互動網頁。

#### 📚 學習重點
- **地震規模與震級**：
  - 規模（Magnitude）：地震釋放的能量
  - 震級（Intensity）：地震對特定地點的影響
  - 規模 vs. 震級的差異
- **建物效應**：
  - 不同建物結構對地震的反應
  - 共振效應（Resonance）
- **土壤放大效應**：
  - 不同土壤類型對地震波的放大效應
  - 土壤液化（Liquefaction）
- **工具**：
  - IRIS Magnitude/Intensity 資源
  - Vercel（部署網頁）
  - GAS（Google Apps Script）

#### 🛠️ 實作任務（作業5）
- [ ] 完成 GitHub Pages 或 Vercel 互動網頁。
- [ ] 繪製一張「規模 vs. 震級」的比較表。
- [ ] 完成一段建物效應或土壤放大效應的說明。
- [ ] 提出一個教學互動問題。
- [ ] 撰寫 200 字防災教學心得。

#### 💡 學習建議
- 如果對網頁開發不熟悉，可以先學習 HTML、CSS、JavaScript 的基礎。
- 多嘗試 IRIS 的 Magnitude/Intensity 資源，了解不同地震的規模和震級。
- 思考如何將地震防災的概念簡化，讓中小學生能理解。

## 💡 學習技巧

### 1️⃣ **主動學習**
- **提前預習**：在課前閱讀相關資料，了解即將學習的主題。
- **課後複習**：在課後整理筆記，複習當天學習的內容。
- **多練習**：工具（如 Git、Python、GAS）需要多練習才能熟悉。

### 2️⃣ **合作學習**
- **小組討論**：積極參與小組討論，分享你的想法，並學習同學的經驗。
- **尋求幫助**：如果遇到問題，不要害羞，主動向老師或同學尋求幫助。
- **分享資源**：如果你找到有用的資源（如教學影片、工具教學），可以分享給同學。

### 3️⃣ **教學應用**
- **時刻思考教學**：在學習每個主題時，時刻思考「如何把這個知識或工具帶回學校教學」。
- **設計教學活動**：每週的作業都要以教學應用為核心，設計適合中小學生的活動。
- **收集教學資源**：收集有用的教學資源（如 IRIS 的活動、地震資料、工具教學），以便日後使用。

### 4️⃣ **持續學習**
- **課後實踐**：課程結束後，持續使用學到的工具和知識，設計更多教學活動。
- **參加社群**：加入地震學或教學相關的社群（如 Facebook 群組、Discord 社群），持續學習和分享。
- **追蹤新知**：關注地震學和教學工具的最新發展，持續更新你的知識。
